th:block

HTML 출력 th:text, th:utext, th:value

        서버에서 넘기는 값을 html 태그에 표현할 때 th:text , th:utext 사용 가능.
        th:text는 html 태그를 단순 문자열로 처리하지만,
        th:utext는 html 태그를 태그 자체로 인식 해준다.
        th:value는 태그의 value 값을 지정할 수 있다.
